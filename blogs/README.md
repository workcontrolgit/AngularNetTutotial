# Blogs

This folder contains supplemental blog posts designed for publication on Medium.com to highlight unique features and developer productivity aspects of this tutorial.

## Purpose

These blog posts serve to:
* **Highlight unique features** of the CAT pattern tutorial (e.g., git submodule workspace approach)
* **Focus on developer productivity** and practical benefits
* **Drive traffic to the main tutorial** via embedded GitHub links
* **Provide bite-sized, shareable content** for the developer community

## File Naming Convention

* **Blog posts:** `{topic}-MEDIUM.md` (e.g., `git-submodule-workspace-MEDIUM.md`)
* **Template:** `BLOG-TEMPLATE.md`

## Publishing Workflow

1. Write blog post following the template
2. Copy entire content to clipboard
3. Paste into Medium editor (medium.com/new-story)
4. Replace placeholder image with actual diagram
5. Preview and adjust formatting
6. Add tags from bottom of article
7. Publish or save as draft
